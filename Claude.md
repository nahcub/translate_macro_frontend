# Claude.md

## 프로젝트 개요

WebSocket 기반 번역 및 CEFR 레벨 분석 서비스

## 주요 기능

1. **실시간 번역**: 단어 및 문장 번역 요청/응답
2. **CEFR 난이도 분석**: 페이지 내 단어들의 난이도 분석
3. **피드백 시스템**: 번역 및 난이도 오류 보고
4. **학습 진행도 추적**: 사용자 학습 완료 단어 기록

## 파일 구조

- `index.js` - 메인 애플리케이션 파일
- `popup.html` - 브라우저 확장 팝업 UI
- `popup.js` - 팝업 로직
- `manifest.json` - 브라우저 확장 매니페스트
- `통신프로토콜.md` - WebSocket 통신 프로토콜 명세
- `backend/` - 백엔드 서버 코드

## 통신 프로토콜

자세한 내용은 [통신프로토콜.md](통신프로토콜.md) 참조

### 프레임 타입
- `0x01`: 번역 요청/응답
- `0x02`: CEFR 레벨 분석 요청/응답
- `0x03`: 번역 피드백
- `0x04`: 난이도 피드백
- `0x05`: 학습 완료 보고

## 개발 노트

- 샘플링 레이트: 100ms (50ms)당 1개 요청
- 엔디안: Big Endian
- 모든 요청에 UUID 포함
- Binary WebSocket 프레임 사용 (opcode 0x2)
